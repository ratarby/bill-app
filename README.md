# bill-app

## demarrer le back

```text
Démarrer le serveur

1 - cd Billed-app-FR-Back-main

2 - npm run run:dev

```

## demarrer le front

```text
Démarrer 'live-server'

1 - cd Billed-app-FR-Front-main

2 - live-server
```

```text
Démarrer le test 

1 - cd Billed-app-FR-Front-main

2 - npm run test
```
